# ⚙️ Diretório: /scripts

## Propósito

Este diretório contém scripts utilitários que ajudam na instalação, desenvolvimento e manutenção do projeto. Eles não são a aplicação principal, mas sim ferramentas de suporte.

## Arquivos Esperados

* **Shell Scripts:** `.sh` para tarefas como instalar dependências do sistema (ex: `install_deps.sh`).
* **Scripts de Calibração:** `.py` para ajudar o usuário a calibrar o ganho e a posição dos microfones (ex: `calibrate.py`).
* **Scripts de Conveniência:** `.sh` ou `.py` para iniciar e parar serviços, formatar código ou rodar testes (ex: `run_robot.sh`).
